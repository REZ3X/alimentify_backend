Authentication

Get Google Auth URL

GET
/api/auth/google

Initiates Google OAuth 2.0 authentication flow.

Response:

{
  "auth_url": "https://accounts.google.com/o/oauth2/v2/auth?..."
}
Public endpoint - No authentication required

Google OAuth Callback

GET
/api/auth/google/callback

Handles OAuth callback and creates/logs in user.

Query Parameters:

code OAuth authorization code from Google

Response:

// For new users:
Redirect to: {FRONTEND_URL}/auth/check-email?email={email}

// For existing verified users:
Redirect to: {FRONTEND_URL}/?token={jwt_token}

Verify Email

GET
/api/auth/verify-email

Verifies user email and returns JWT token for auto-login.

Query Parameters:

token Email verification token

Response:

{
  "success": true,
  "message": "Email verified successfully",
  "token": "jwt_token_here",
  "user": {
    "id": "user_id",
    "email": "user@example.com",
    "name": "User Name"
  }
}

Get Current User

GET
/api/auth/me

Retrieves authenticated user information.

Headers:

Authorization: Bearer {jwt_token}

Response:

{
  "id": "user_id",
  "email": "user@example.com",
  "name": "User Name",
  "picture": "profile_picture_url"
}
Protected endpoint - Requires authentication

Logout

POST
/api/auth/logout

Logs out the current user and invalidates session.

Headers:

Authorization: Bearer {jwt_token}

Response:

{
  "success": true,
  "message": "Logged out successfully"
}
Protected endpoint - Requires authentication

Debug Configuration

DEV ONLY GET
/api/auth/debug-config

Returns sanitized OAuth configuration for debugging purposes. Use only in development.

Response:

{
  "client_id": "your-client-id",
  "redirect_uri": "http://localhost:4000/api/auth/google/callback",
  "frontend_url": "http://localhost:3000"
}
Development only - Should be disabled in production

Health Profile

Create or Update Profile

POST
/api/health/profile

Creates or updates user health profile and calculates daily nutrition targets.

Headers:

Authorization: Bearer {jwt_token}
Content-Type: application/json

Request Body: 

{
  "age": 25,
  "gender": "male",
  "height_cm": 175,
  "weight_kg": 70,
  "activity_level": "moderately_active",
  "goal": "maintain_weight"
}

Activity Levels:

sedentary - Little or no exercise
lightly_active - Light exercise 1-3 days/week
moderately_active - Moderate exercise 3-5 days/week
very_active - Hard exercise 6-7 days/week
extremely_active - Very hard exercise & physical job

Goals:

lose_weight - 500 calorie deficit
maintain_weight - Maintenance calories
gain_weight - 500 calorie surplus

Response:

{
  "success": true,
  "profile": {
    "age": 25,
    "gender": "male",
    "height_cm": 175,
    "weight_kg": 70,
    "activity_level": "moderately_active",
    "goal": "maintain_weight",
    "daily_calories": 2500,
    "daily_protein_g": 140,
    "daily_carbs_g": 281,
    "daily_fat_g": 83
  }
}
Protected endpoint - Requires authentication

Meal Tracking & Analytics

Log meals, track daily nutrition, and analyze long-term trends with comprehensive period statistics including goal progress and compliance tracking.

Log Meal

POST
/api/meals/log

Logs a meal entry with nutritional information.

Headers:

Authorization: Bearer {jwt_token}
Content-Type: application/json

Request Body: 

{
  "meal_type": "breakfast",
  "food_name": "Scrambled Eggs",
  "calories": 250,
  "protein_g": 18,
  "carbs_g": 2,
  "fat_g": 20,
  "serving_size": "2 eggs",
  "notes": "With butter"
}

Meal Types:

breakfast, lunch, dinner, snack

Response:

{
  "success": true,
  "meal": {
    "id": "meal_id",
    "user_id": "user_id",
    "meal_type": "breakfast",
    "food_name": "Scrambled Eggs",
    "calories": 250,
    "protein_g": 18,
    "carbs_g": 2,
    "fat_g": 20,
    "serving_size": "2 eggs",
    "notes": "With butter",

}

AI Nutrition Analysis

Analyze Food Image

POST
/api/nutrition/analyze

Analyzes food image using Google Gemini AI to extract nutritional information.

Headers:

Authorization: Bearer {jwt_token}
Content-Type: multipart/form-data

Form Data:

image Image file (max 10MB, JPEG/PNG)

Response:

{
  "success": true,
  "analysis": "{\n  \"food_name\": \"Cheeseburger\",\n  \"serving_size\": \"1 burger\",\n  \"calories\": \"600-700\",\n  \"macronutrients\": {\n    \"protein\": \"30-35g\",\n    \"carbohydrates\": \"45-50g\",\n    \"fat\": \"35-40g\"\n  },\n  \"health_score\": 5,\n  \"health_notes\": \"...\",\n  \"recommendations\": \"...\"\n}",
  "timestamp": "2025-01-01T12:00:00Z"
}
Protected endpoint - Requires authentication
